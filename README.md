# Grunt Website

## Local Development

1. Download dependencies:
   ```shell
   npm install
   ```
2. Create production site:
   ```shell
   npm run build
   ```
3. Start web server:
   ```shell
   npm start
   ```

Local address is: http://localhost:5678/

To automatically rebuild and restart the server after any file changes,
use the development mode instead:

```shell
npm run dev
```
